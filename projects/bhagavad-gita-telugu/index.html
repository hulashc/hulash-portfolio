Why the Bhagavad Gita Endures

  • →A 5,000-year-old dialogue between Arjuna and Krishna — 700 verses on duty, action, and the self.
  • →The concept of Nishkama Karma — act without attachment to outcomes — influenced thinkers from Thoreau to Gandhi.
  • →The battlefield is a metaphor for inner conflict — the war between duty and desire every person faces.
  • →18 chapters, each a distinct path — Karma Yoga (action), Jnana Yoga (knowledge), Bhakti Yoga (devotion).
  • →This project surfaces that wisdom live via API — making ancient text accessible and programmable for the modern web.
